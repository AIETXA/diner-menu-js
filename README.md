## Diner Menu con JS

>Este en un ejercicio práctico en el que tendré que crear el menu de un restaurante con JavaScript


Restaurante Bottega.
Incluye un menú principal y un menú de acompañamientos.
Se ofrece un plato principal y dos acompañamientos a precio regular.

### Instrucciones:
- Muestra el menú completo (impreso)
- El usuario selecciona un plato principal.
- La camarera hace un comentario basado en su selección
- El comentario puede ser una comparación de los dos platos o un comentario aleatorio extraído de un archivo de comentarios.
- Indica el precio.
- Repite las opciones anteriores para los acompañamientos (comentario y precio).
- Calcula el total.

#### 💡 EXTRA:
Disponemos de menús para el desayuno, el almuerzo y la cena. El desayuno tiene diferentes opciones, mientras que el almuerzo y la cena ofrecen las mismas opciones, pero a precios diferentes.

#### 💡 EXTRA: 
Permite la personalización de los artículos (cómo se preparan, decide las implicaciones de costos adicionales).
